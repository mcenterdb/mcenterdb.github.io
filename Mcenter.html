API Overview

Our API is available for everyone to use. A TMDb user account is required to request an API key. Professional users are approved on a per application basis. As always, you must attribute TMDb as the source of your data. Please be sure to read the API FAQ. If you're building a product or service around TMDb, we also recommend joining our Google+ community. It's the best place to learn about API changes, new features and deprecation notices.
